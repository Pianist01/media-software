Adding Song:

- Title
- Author
- Language
- Type: Praise or Worship
- Verse
- Chorus
- Verse
- Bridge 
- Chorus

Song:

- Title
- Author
- Type
- Language
- Lyrics
- How often used

Example:

Title: Padre Nuestro
Author: Bethel Music
Language: Spanish
Type: Worship
Verse: "Padre nuestro en los cielos, tu nombre santo es..."
Bridge: "El cielo aqui, el cielo aqui..."
Chorus: "Tuyo es el reino tuyo, el poder tuya, es la gloria por siempre amen."
User can then add more verses, or remove verses. User can also change lyric type. Default will be:

Title, Author, Language, Type, Verse, Chorus, Verse, Bridge, Chorus. 

However, user will be able to change each however they want. It will be a drop down menu next to each lyric where user can label them as verse, chorus, etc.
