Questions to answer for this project.

1. What problem am I solving? Current media person at church as expressed frustrations with current software we use for church. We are a small church, so we don't have the funds to be able to purchase the expensive ones. I want to make a high quality app where users can display lyrics, Bible verses, images, videos, and songs. However, I also want user to be able to have freedom to customize as much as he/she wants. Fonts, themes, etc. 

2. Who uses this? My target audience are churches. My hope is to make an easy to understand app, where anyone can hop in and be able to create a high quality service without having to go through so many buttons/options.

3. What features are required? App must allow user to add and store lyrics so that each time a new service is created, the previous songs/lyrics used are already saved. They must be able to display lyrics, Bible verses, videos, and play music in background if needed. There should be a list of default popular songs already there for users, both in Spanish and English. Users should be able to customize themes as they wish, with default themes already there for them, but also need to be able to upload their own theme. Text should be customizable as well, with different fonts, sizes, etc.

4. How should songs be stored? Songs should be stored in a list called: "Saved songs". So when user adds own song or selects song, user will have option to save song in their own collection. They can then always pick from that collection instead of having to search song again. Collection should also have filters that user can add such as "A-Z, Latest, Oldest, etc."

5. How will projector mode work? There will be preview screeen and live screen user can see. Any changes user makes will show up on preview screen. That way, they can see how changes look before pushing them to live screen. When HDMI cable is connected to computer, button that says, "Display Screen" will become available under or next to live screen. Once button is clicked, whatever is shown on live screen will displayed on projector. 

6. How will dual-screen mode work? When projector or TV displays what is on live screen, that is all that will be shown. User will see the app hud with all available options to make changes, but TV or Projector will only every show live screen and nothing else. 

7. What is version 0.1? I'd like this first version to just make sure that features are actually functioning. Saving songs, adding verses, etc.
